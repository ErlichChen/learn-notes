# Git Flow

## config user

```bash
git config --global user.name 'your_name'
git config --global user.email 'your_email@domain.com'
```

## config scope

```bash
git config --local
git config --global
git config --system

# show config
git config --list --local
git config --list --global
git config --list --system
```

## initial git rep

```bash
git init
```

## commit

```bash
git add .
git add 'file name'
```
