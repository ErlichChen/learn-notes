Box display types

A block box is defined as a box that's stacked upon other boxes, and can have width and height set on it. The whole box model as described above applies to block boxes.

An inline box is the opposite of a block box. it flows with the document's text. Width and height settings have no effect on inline boxes; any padding, margin and border set on inline boxes will update the position of surrounding text, but will not affect the position of surrounding block boxes.

An inline-block box is something in between the block and inline. It flows with surrounding text without creating line breaks before and after it like an inline box, but it can be sized using width and height and maintains its block integrity like a block box.
